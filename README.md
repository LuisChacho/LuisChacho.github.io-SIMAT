# 🎓 Simulador de Matemáticas - 200 Preguntas de Admisión

Plataforma educativa interactiva autocontenida que simula exámenes de admisión mediante un banco robusto de **200 preguntas reales** sin repetición y con soporte de editor matemático.

## 📦 Componentes Incluidos
1. `index.html`: Maquetación estructurada y accesible.
2. `index.css`: Hoja de diseño estilo plataforma universitaria con soporte tipográfico para editor de ecuaciones nativo (`sup`, `sub`, fuentes serif tipificadas).
3. `index.js`: Lógica basada en el algoritmo de aleatoriedad Fisher-Yates y gestión dinámica de comodines (50/50, Eliminar Uno, Llamada y Público).
4. `README.md`: Manual operativo de despliegue.

## ⚙️ Configuración y Despliegue en GitHub Pages
1. Sube los 4 archivos a la rama raíz (`main`) de un repositorio público en tu perfil de GitHub.
2. Accede a **Settings** -> **Pages** dentro del panel del repositorio.
3. Elige la rama `main` y haz clic en **Save**.
4. La URL pública del simulador estará lista para tus estudiantes en menos de 2 minutos.
