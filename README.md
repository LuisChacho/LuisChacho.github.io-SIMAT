# Plataforma Modular de Evaluación - Versión Corporativa Limpia

Arquitectura de software desacoplada, optimizada estéticamente con los requerimientos formales institucionales.

## Cambios Implementados:
1. **Tipografía Formal:** Expresiones libres de caracteres planos confusos (`^`, `*`, `/`). Uso estricto de superíndices algebraicos nativos (`x¹⁰`, `y⁻²`, `·`).
2. **Eliminación de Prefijos:** El sistema ya no inyecta cadenas pesadas como "Pregunta N° 1 -" o "Reactivo N° X". El estudiante visualiza únicamente el enunciado directo del reactivo matemático.
3. **Remoción de Tiempo Extremo:** Se removió el temporizador de cuenta regresiva global para fomentar la concentración limpia y fluida del evaluado.
4. **Diseño Premium Corporativo (Estilo Empresa):** Layout limpio con fondo de página blanco/grisáceo suave (`#f8fafc`), tarjetas ordenadas sobre contenedores blancos puros, botones con azul institucional sutil y tipografía minimalista de alta legibilidad.
